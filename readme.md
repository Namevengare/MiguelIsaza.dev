### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Namevengare/MiguelIsaza.dev
   cd MiguelIsaza.dev
   ```

2. **Install dependencies**
   ```bash
   npm i
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   - Navigate to `http://localhost:3000`
